## โมดูลนี้ครอบคลุมอะไรบ้าง

โมดูลนี้พาคุณจากการติดตั้ง Keycloak ที่ทำงานได้ ไปสู่ระบบที่ทำงานอัตโนมัติ มีแบรนด์ของตัวเอง ตรวจสอบได้ และพร้อมสำหรับ production สี่เสาหลักได้แก่:

- **Admin REST API** — ทุกสิ่งที่คุณทำในหน้าจอ admin console นั้นขับเคลื่อนด้วย REST API คุณสามารถทำให้การตั้งค่า realm, การสร้างผู้ใช้ และการกำหนดค่า client เป็นอัตโนมัติจาก script หรือ CI pipeline แทนที่จะต้องคลิกผ่าน UI ทุกครั้ง
- **Themes** — Keycloak แสดงผลหน้า login, account และ admin ของตัวเอง คุณสามารถแทนที่ด้วย template ของคุณเองเพื่อให้ตรงกับแบรนด์
- **Events & audit** — Keycloak บันทึก login event และ admin event เปิดใช้งานก่อน go live เพื่อให้มี audit trail เมื่อมีสิ่งผิดปกติ
- **Production hardening** — เปลี่ยนจาก `start-dev` ไปยังโหมด `start` ที่ optimised พร้อม external database, hostname ที่กำหนด และการจัดการ HTTPS อย่างถูกต้อง
