## default value กับ override rules

ทุก Feature ใน GrowthBook มีการ resolve ค่าสองชั้น:

1. **Override rules** — เงื่อนไข, percentage rollout หรือการกำหนด experiment ที่คุณตั้งค่าต่อ environment เมื่อ rule ตรงกับ context ของผู้ใช้ปัจจุบัน rule นั้นจะชนะ
2. **Default value** — ค่าที่คืนกลับมาเมื่อ **ไม่มี rule ตรงเงื่อนไข** นี่คือพื้นที่ปลอดภัยของคุณ

เมื่อ GrowthBook เข้าถึงไม่ได้เลย (network error, CDN ล่ม) SDK จะตกลงไปใช้ **argument สำหรับ fallback** ที่คุณส่งโดยตรงในโค้ดของคุณ — มันไม่เคยคืน `null` หรือ throw

## fallback สามชั้น

```
SDK call
  └─ GrowthBook loaded?
       ├─ YES → evaluate rules → match found? → return rule value
       │                       └─ no match  → return Feature default value
       └─ NO  → return the fallback argument from your code
```

## ส่ง fallback ที่สมเหตุสมผลเสมอ

```js
// BAD — if GrowthBook is unreachable, getFeatureValue returns undefined
const text = gb.getFeatureValue('banner-text');

// GOOD — safe fallback keeps the app working
const text = gb.getFeatureValue('banner-text', 'Welcome');
```

argument สำหรับ fallback คือแนวป้องกันด่านสุดท้าย มันควรแทนประสบการณ์ที่คุณต้องการให้ผู้ใช้ได้รับหาก GrowthBook ล่ม — โดยปกติคือพฤติกรรมปัจจุบันที่เสถียร

## ตั้งค่า default value ใน GrowthBook

1. เปิด Feature ใน GrowthBook UI
2. คลิก **Edit** ข้าง ๆ ฟิลด์ **Default Value**
3. ใส่ค่า เช่น `Welcome` สำหรับ flag แบบ string
4. คลิก **Save**

default value จะมีผลทันทีสำหรับทุก environment ที่ไม่มี override rule ใดให้ผลลัพธ์ที่ตรงเงื่อนไข

## เปลี่ยน default value

ในการอัปเดต default value โดยไม่เพิ่ม rule:

1. เปิด Feature
2. คลิก **Edit** บน Default Value
3. เปลี่ยนค่าแล้วคลิก **Save**

การเปลี่ยนแปลงจะถูกนำไปใช้ทั่วทุก environment ที่ไม่มี rule แบบ override โดยไม่ต้อง deploy

## ตัวอย่างใช้งานจริง — banner text

```js
import { GrowthBook } from '@growthbook/growthbook';

const gb = new GrowthBook({
  apiHost: 'https://cdn.growthbook.io',
  clientKey: process.env.GROWTHBOOK_CLIENT_KEY,
  attributes: {
    id: 'user-123',
    country: 'TH',
  },
});

await gb.init({ timeout: 2000 });

// Returns the GrowthBook default value when no rule matches,
// or 'Welcome' when GrowthBook cannot be reached.
const bannerText = gb.getFeatureValue('banner-text', 'Welcome');

document.getElementById('banner').textContent = bannerText;
```
