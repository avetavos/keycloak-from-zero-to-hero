SDK Connection คือวิธีที่คุณอนุญาตให้แอปของคุณดึง feature flags จาก GrowthBook แต่ละ connection ถูกจำกัดขอบเขตไว้ที่ environment หนึ่งและ language/framework หนึ่ง

## การสร้าง SDK Connection

1. ใน GrowthBook UI ไปที่ **Settings** ใน sidebar ด้านซ้าย
2. คลิก **SDK Connections**
3. คลิก **Add SDK Connection**
4. ตั้งชื่อให้มัน (เช่น "Node.js Dev")
5. เลือก **Environment** (เช่น `development`)
6. เลือก **Language / Framework** (เช่น JavaScript/Node.js)
7. คลิก **Save** GrowthBook จะแสดงค่าสองค่าให้คุณ:
   - **apiHost** — สำหรับ self-hosted: `http://localhost:3100`; สำหรับ Cloud: `https://cdn.growthbook.io`
   - **clientKey** — string ที่ขึ้นต้นด้วย `sdk-...`
8. คัดลอกทั้งสองค่า คุณจะใช้มันในการ initialize SDK

```js
import { GrowthBook } from '@growthbook/growthbook';

const gb = new GrowthBook({
  apiHost: 'http://localhost:3100',   // self-hosted SDK API
  clientKey: 'sdk-YOUR_CLIENT_KEY',
});

await gb.init({ timeout: 2000 });

// Check a boolean flag
if (gb.isOn('my-feature')) {
  console.log('Feature is enabled');
}
```

<Callout title="Key point">
SDK Connection จะจำกัดขอบเขต clientKey หนึ่งตัวไว้กับ environment ที่เฉพาะเจาะจง ใช้ SDK Connection แยกกัน (และดังนั้นจึงมี clientKey แยกกัน) สำหรับ development กับ production เพื่อให้สถานะของ flag แยกขาดจากกัน
</Callout>
