หลังจากเริ่ม GrowthBook ด้วย Docker แล้ว สิ่งแรกที่ต้องทำคือสร้างบัญชี admin และตั้งค่า organization ของคุณ

1. เปิด `http://localhost:3000` (หรือ Cloud URL ของคุณ) ใน browser
2. คลิก **Get Started** หรือ **Register** — ระบบจะให้คุณสร้างบัญชี admin บัญชีแรก กรอกชื่อ, อีเมล และรหัสผ่านของคุณ
3. สร้าง **Organization** ของคุณ — ตั้งชื่อให้มัน (เช่น "My Company") นี่คือ container ระดับบนสุดสำหรับทุกอย่างใน GrowthBook
4. (ไม่บังคับ) สร้าง **Project** — Project ช่วยให้คุณจัดกลุ่ม features และ experiments ภายใน organization คลิก **Create Project** และตั้งชื่อให้มัน
5. ตรวจสอบ **Environments** เริ่มต้น: GrowthBook สร้าง `development`, `staging` และ `production` ให้โดยอัตโนมัติ คุณสามารถเพิ่มหรือเปลี่ยนชื่อได้ที่ **Settings → Environments**
6. สำรวจ sidebar ด้านซ้าย:
   - **Features** — feature flags ของคุณ
   - **Experiments** — A/B tests
   - **Metrics** — นิยามว่าความสำเร็จหน้าตาเป็นอย่างไร
   - **Settings** — SDK connections, environments, สมาชิกทีม, data sources

<Callout title="Key point">
บัญชีแรกที่คุณสร้างจะกลายเป็น admin ของ organization ไม่มีขั้นตอนสร้าง admin แยกต่างหาก — ใครก็ตามที่ register เป็นคนแรกจะเป็นเจ้าของ organization
</Callout>
