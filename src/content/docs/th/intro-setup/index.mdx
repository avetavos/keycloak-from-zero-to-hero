## Keycloak คืออะไร?

Keycloak เป็น open-source Identity and Access Management (IAM) server ที่จัดการ authentication, Single Sign-On (SSO) และ authorization ให้กับแอปพลิเคชันของคุณด้วย standard protocols ได้แก่ OpenID Connect (OIDC), OAuth2 และ SAML แทนที่จะสร้าง login form, password hashing, token handling และ MFA ในทุก app คุณเพียงแค่ delegate งานเหล่านี้ให้กับ Keycloak

## วงจรหลัก

ทุก workflow ใน Keycloak ทำตาม loop สี่ขั้นตอนเดียวกัน:

1. รัน Keycloak ด้วย Docker
2. เปิด admin console
3. สร้าง realm และลงทะเบียน client
4. Secure แอปของคุณด้วย OIDC
