## GrowthBook คืออะไร?

GrowthBook คือ **แพลตฟอร์มแบบ open-source** ที่รวมสามความสามารถไว้ในเครื่องมือเดียว:

- **Feature flags** — ควบคุมโค้ดทุกเส้นทางไว้หลัง toggle ที่ตั้งชื่อได้และสั่งงานจาก UI ได้เอง
- **A/B testing และ experimentation** — รัน experiment ที่มีความเข้มงวดทางสถิติได้โดยไม่ต้องเขียนโค้ดวิเคราะห์ผล
- **Analytics บน data warehouse ของคุณเอง** — เชื่อมต่อ Snowflake, BigQuery, Redshift หรือ SQL source ใดก็ตามที่คุณมีอยู่แล้ว โดย GrowthBook ไม่เคยแตะ raw events ของคุณเลย

คุณสามารถ **self-host** บน infrastructure ของคุณเอง (Docker, Kubernetes) หรือใช้ **GrowthBook Cloud** ที่ไม่ต้องดูแล server เลยก็ได้ คอร์สนี้ใช้ Docker เพื่อให้ทุกแนวคิดสามารถทำซ้ำได้บนเครื่องของคุณเอง

## Core loop

ทุก workflow ของ GrowthBook จะมีสี่ขั้นตอน:

1. **รัน GrowthBook** — เริ่มแพลตฟอร์ม (Docker หรือ Cloud)
2. **ล็อกอิน** — สร้างบัญชี admin ของคุณและสำรวจ dashboard
3. **สร้าง feature flag** — กำหนด key, type และค่า default ให้กับมัน
4. **อ่านค่า flag ในแอปของคุณ** ผ่าน SDK — แค่เรียก `isOn()` ครั้งเดียวก็พอ

โค้ดด้านล่างแสดงการตั้งค่า SDK ขั้นต่ำที่คุณจะค่อย ๆ สร้างขึ้นจนถึงในบทเรียนที่ 5

export const firstTasteCode = `import { GrowthBook } from "@growthbook/growthbook";

const gb = new GrowthBook({
  apiHost: "https://cdn.growthbook.io",
  clientKey: "sdk-abc123",          // from SDK Connection page
});

await gb.loadFeatures();

if (gb.isOn("dark-mode")) {
  enableDarkMode();
}`;
